# Helm Sandbox

This repository is a Helm learning sandbox to understand how Helm is used in real projects.

## Repository Overview

### Using Existing Helm Packages
This folder shows how to:
- Use existing Helm charts
- Add a Helm repository
- Install applications using Helm

### Packaging a Custom Application
This folder focuses on:
- Creating a custom Helm chart
- Updating values.yaml
- Packaging the application
- Deploying it using Helm

### Applying Helm in a Real-World Application
This section demonstrates:
- Deploying a real application using Helm
- Managing configuration through Helm values
- Installing and upgrading releases

## Goal of This Repo
- Learn Helm in a practical way
- Understand real-world Helm usage
- Practice packaging and deploying applications
